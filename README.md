# primal
Prosty CMS z bazą danych w pliku JSON.

## Działa
* Edycja bloków WYSIWYG na stronie, zmiana <title>, meta description
* Przyjazne linki
* Szablony wewnątrz plików .php (block name="top") generuje blok WYSIWYG na froncie
* Pamięć podręczna
